1. 개요
1.1. 목적
   도서관에 없는데 읽고 싶은 책이 있을 때 신청하기 위한 목적
1.2. 대상
   청주대학교 학생

3. 프로그램의 중요성 및 필요성
2.1 중요성
   책이 필요한 학생에게는 중요함
2.2 필요성
   책이 필요한 학생에게 필요함
